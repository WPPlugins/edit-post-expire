Prohibit authors edit published posts after a specified time interval

== Description ==

This plugin allows to prohibit authors edit their published posts after a specified time interval, just like in forums, where your post could be blocked for editing after 5 minutes.
Visit http://www.shinephp.com/edit-post-expire-wordpress-plugin/ for more details.


== Installation ==

Installing procedure:

1. Deactivate plugin if you have the previous version installed. (It is important requirement for switching to this version from a previous one.)
2. Extract "edit-post-expire.zip" archive content to the "/wp-content/plugins/edit-post-expire" directory.
3. Activate "Edit Post Expire" plugin via 'Plugins' menu in WordPress admin menu. 
4. Go to the "Settings"-"Edit Post Expire" menu item and setup time interval in minutes after which you wish to block published posts for editing by their authors.
